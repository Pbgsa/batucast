Feature: Gerar link para páginas
	As: um usuário do batucast
	I want to: gerar links para faixas específicas ou playlists
	So that: eu possa compartilhar faixas específicas ou playlists com outras pessoas.

	Cenário: Gerar link para Faixa Específica
		Given: Usuário de nome “Arnaldo” se encontra na página da faixa “Abacaxi de Irará”
		When: Ele solicita a criação de um link de acesso para a faixa “Abacaxi de Irará” 
		Then: O link para a faixa “Abacaxi de Irará” será copiado para a área de transferência de Arnaldo

	Cenário: Falha na Disponibilidade da Faixa:
		Given: Usuário de nome “Arnaldo” se encontra na página da faixa “Abacaxi de Irará”.
		And: Em sua área de transferência está um link para a faixa "Wire Sponge Theme".
		When: Ele solicita a criação de um link de acesso para a faixa “Abacaxi de Irará”.
		And: A faixa se econtra temporariamente indisponível no sistema.
		Then: Não será gerado um link para a faixa "Abacaxi de Irará".
		And: Na área de transferência de "Arnaldo" estará um link para a faixa "Wire Sponge Theme".
